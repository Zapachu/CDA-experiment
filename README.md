# Experiment Link Framework

## 项目结构
  - bespoke ： 定制实验
  - component : 前端组件包
  - gameTrial : 定制实验体验入口
  - linker ：调度管理模块
  - protocol ：系统间数据交换协议(RPC接口封装,Schema结构声明等)
  - proxy
    - thirdPartyPhase : 与bespokeGame平级,向linker注册所代理的第三方页面，详见thirdPartyPhase/README.MD
